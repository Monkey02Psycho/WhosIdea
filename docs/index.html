What is forge?

Forge is an open source tool to program mods for Minecraft. Forge removes obfuscation from Minecrafts code and provides tools so developers can add more content to the game. Forge also acts as a loader to load code from the mods into the game. Forge is not officially supported by Mojang(The owners of Minecraft) so all of forge and modding in general is community ran

Basic Structure of a Mod

This is roughly the basic structure of a mod. The src folder contains all the packages for the mod. The main package is the main folder of the mod. inside that are two sub folders. java and resources. Inside java is where all the code is located. in resources is where all of the various resources are held. I will go more in depth in these in a later section
